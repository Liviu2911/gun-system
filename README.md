# Gun System

Hello, this is a Roblox portfolio project, written entirely in luau, in vscode (through rojo). I can, of course use RobloxStudio as well.
I used raycasts and a state manager for the gun system, to be able to decide which ammo, for what weapon should be displayed, when to zoom in, 
when the player can reload, or not, and so on.

### Other systems included
This project embbeds Datastores and a scalable inventory system as well.
